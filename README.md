- Скрипт предназначен для блокировки "тупого" ддоса по одному и томуже URL, при условии что в качестве web сервера используется nginx
- Для работы скрипта необхоимо что бы на сервере был установлен firewalld и ipset, так же скрипт создаёт файл с обнаруженными ip адресами с которых идёт атака для того что бы можно было заблокировать их вручную другим способом.
- Скрипт работает только с дефолтным форматом логов nginx.
- Скрипт проеряет доступность URL чреез curl запрос. URL страницы которую необходимо проверять и путь к файлу логов nginx нужно настроить в параметрах класса вручную.
